Data Binding

AngularJS creates live templates as a view. Views are interpolated when the view is evaluated with one or more variable substitu- tions; the result is that the variables in our string are replaced with values.

Angular runs its own event loop to check if a value is dirty. A value is considered dirty if it has changed since the last time the event loop ran. This is how Angular is able to track and respond to changes in the app. The event loop is called the $digest() loop

To denote internals and other built-in library functions, Angular uses the $ to prepend objects.
