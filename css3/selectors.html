Difference Between Adjacent and General Sibling Cominators

Adjacent Sibling selects any element ( F ) that is immediately preceded by element ( E ) on the same level of the document tree.
General Sibling selects any element ( F ) that is preceded by element ( E ) on the same level of the tree, regardless of whether it is immediately adjacent.
