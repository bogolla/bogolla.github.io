HTML5

HTML5 is a markup language used for structuring and presenting content for the World Wide Web. HTML5 aims have to improve the language with support for the latest multimedia while keeping it easily readable by humans and consistently understood by computers and devices (web browsers, parsers, etc.). It is intended to subsume HTML 4 and XHTML 1

SYNTAX

HTHL5 Syntax can be summarised with the following points:

  • White spaces - Not significant because browsers and validator don't require it to render pages.
  • Not case sensitive - In HTML5 lowercase, uppercase... anything goes.

Audio

Embedding an audio file in an HTML5 document is simple:

  • audio tags embed audio with native browser controls to web pages.
  • Audio API - gives access to methods such as play and pause and properties such as volume .

Video

Adding video to a page is equally simple:

  • The audio tag has native default controls.
  • The location of the video file can be specified by either using the src attribute on the video element or by using source elements nested within the opening and closing video tags.
  • Images or posters can be used as placeholders for the videos
